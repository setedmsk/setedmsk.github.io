02 Web App PWA + Mobile

PingScope

Ferramenta de rede · Ping em tempo real · App mobile-first

O desafio

Criar uma ferramenta simples para qualquer usuario testar se um IP ou dominio esta respondendo, sem depender de comandos no terminal e com leitura rapida do status.

A solucao

App com interface mobile-first, historico de testes, modo automatico e backend em Node.js executando o ping real do sistema. A base tambem ficou preparada para Expo, permitindo usar a mesma API em um app mobile.

Node.js API REST PWA React Native · Expo Network Tool
